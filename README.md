# CloneIt Script

A simple bash script to quickly clone GitHub repositories using `gh` and `fzf` for selection.

## Prerequisites

- GitHub CLI (`gh`) installed and authenticated
- `fzf` for fuzzy search
- `jq` for JSON processing

## Installation

Clone this repository and run the script:

```bash
./cloneit.sh
```

## Usage

1. Run `./cloneit.sh`
2. Use fuzzy search to select a repository
3. The script will clone the selected repository to the current directory

This is designed for Linux Fedora systems.
